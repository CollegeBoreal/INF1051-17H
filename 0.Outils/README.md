# Créer un projet à l'aide d'outils de configuration

## activator

### Liste des projets
```
$ activator list-templates
```

### Créer un projet

```
$ activator new scala101 minimal-scala
```
### Ouvrir le projet avec IntelliJ

Ouvrir le fichier 'build.sbt'

### Utiliser le sbt CLI


```
$ sbt compile
```

```
$ sbt run
```

```
$ sbt test
```
